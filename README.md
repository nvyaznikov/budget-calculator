# Cash Keeper

React-приложение для учета и анализа личных финансов с серверной авторизацией.

## Возможности

- учет доходов, обычных расходов и регулярных платежей;
- редактирование и удаление операций;
- фильтр операций и аналитики по месяцу;
- расчет баланса, дневного бюджета и планируемых накоплений;
- графики доходов и расходов на `Chart.js`;
- экспорт данных в JSON;
- светлая и темная тема;
- JWT-авторизация;
- хранение операций в `SQLite`.

## Запуск

```bash
npm install
npm run dev:all
```

После запуска:

- frontend: `http://localhost:5173`
- API: `http://localhost:3001/api`

Можно запускать отдельно:

```bash
npm run server
npm run dev
```

Production-сборка:

```bash
npm run build
```
